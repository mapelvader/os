

### Instalasi Linux (deb)
```
sudo apt-get install git
sudo apt-get install python3
```

### Instalasi Termux
```
pkg install git
pkg install python3
```

### Copy Tools 
```
git clone https://github.com/mapelvader/os.git
cd os
pip3 install -r requirements.txt
python3 os.py
```
### UNTUK MENJALANKAN
```
cd os
python3 os.py
```

ini Adalah Program Script Profesional Berbasis Private Intelegence Yang Berfokus Untuk Mengambil, Mencari Dan Memvalidasi Data Siswa Sekolah Di Indonesia, Dirancang Untuk Analisis Data Pendidikan Yang Lengkap Dan Akurat

– Dump Database Sekolah Lengkap Berdasarkan Npsn Sekolah Dan Menyediakan Informasi Lengkap Siswa
– Validate Nik Ke Metadata Bpjs
– Validate Ke Data Ptk
– Sistem Pencarian Interaktif Berdasarkan Nama Siswa Yang Ingin Dicari 
– Menyediakan Lokasi Siswa Lengkap Untuk Analisis Data
– Output Mudah Digunakan Dan Dikembangkan
– Data Siswa Tersimpan Rapi Dalam Format Json Dan Csv
– Validate Data Menggunakan Sumber Resmi Dukcapil/Dapodik/Kemendikbud Dll
